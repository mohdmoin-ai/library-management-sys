# Library Management System (Python)

## Overview

Library Management System is a simple command-line Python project that helps manage books in a library. Users can add books, search for books, borrow books, and return books through a menu-driven interface.

This project was built to practice Python fundamentals such as dictionaries, loops, conditions, functions, and user input handling.

## Features

* Add new books to the library
* Search for a book by name
* Borrow available books
* Return borrowed books
* Menu-driven interface
* Simple and beginner-friendly code

## Technologies Used

* Python 3

## Concepts Used

* Variables
* Data Types
* Dictionaries
* Loops (`while`)
* Conditional Statements (`if`, `elif`, `else`)
* Functions
* User Input Handling

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python library_management.py
```

## Example Menu

1. Add Book
2. Search Book
3. Borrow Book
4. Return Book
5. Exit
